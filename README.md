## âœ¨ Features
- Fully **designed in SolidWorks**, including hull geometry, motor mount, rudder, and steering mechanisms  
- Fully **3D-printed hull and custom components**, assembled into a functional RC boat  
- **Waterproofed structure** using epoxy and fiberglass for durability and water resistance  
- **Arduino-based control system** for propulsion and steering using servo motors

  - **ESC-based system** (final implementation)
